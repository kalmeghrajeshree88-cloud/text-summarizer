# 📝 Text Summarization Chatbox

A text summarization web app built using Hugging Face Transformers and Gradio.

## 🚀 Live Demo
👉 [Click here to try the app](https://huggingface.co/spaces/rajeshree2/text-summarizer)

## 📌 Features
- Summarizes long articles, essays, and reports
- Bullet point mode for point-wise summaries
- Adjustable summary length using sliders
- Word count statistics showing reduction percentage
- Clean chat-style interface

## 🛠️ Technology Stack
| Layer | Technology |
|---|---|
| AI Model | DistilBART (sshleifer/distilbart-cnn-12-6) |
| ML Framework | PyTorch |
| NLP Library | Hugging Face Transformers |
| Web Interface | Gradio |
| Hosting | Hugging Face Spaces |
| Language | Python 3.10 |
